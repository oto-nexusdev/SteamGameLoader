# SteamGameLoader
Gerenciador de Arquivos .manifest e .lua. Integrados com API's públicas de download e compatível com Steam Tools. Compatibilidade e funcionamento estável com Windows &amp; Linux.

CASO A INSTALAÇÃO DO ARQUIVO, MESMO QUE DÊ COMO CONFIRMADA, NÃO ACONTEÇA. DESINSTALE O STEAM TOOLS E REINSTALE NOVAMENTE. ATUALIZAÇÕES DA STEAM CAUSAM ESTE ERRO.
